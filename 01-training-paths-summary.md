## Learning Paths

### Shared Foundation
Build comfort with:
- notebook basics
- Python basics
- reading starter code
- debugging
- data basics
- using AI well

### Classic AI
Focus more on:
- tabular data
- pandas
- patterns
- anomaly-style thinking
- rules and guardrails
- interpreting results

### Agentic AI
Focus more on:
- prompt structure
- tool functions
- context passing
- workflow thinking
- evaluating output quality
