# 13 Glossary

This glossary gives short student-friendly definitions for common terms used in this repository.

## Terms

**Agentic AI**  
An AI system that works through steps, uses instructions, and may call tools to complete a task.

**Anomaly**  
Something unusual compared with the normal pattern.

**API**  
A way for one program to request information from another system.

**Bar chart**  
A chart used to compare categories.

**Cell**  
A block inside a Jupyter notebook that contains either code or text.

**Classic AI**  
A general term here for data-focused workflows such as analysis, prediction, pattern finding, or anomaly-style tasks.

**Column**  
A field in a table. Each column stores one kind of information.

**Context**  
The information a system needs to remember or use in the next step.

**CSV**  
A simple file format used for tables of data.

**DataFrame**  
A table-like structure in pandas with rows and columns.

**Debugging**  
Finding and fixing mistakes in code or workflow steps.

**Feature**  
A piece of information used by a model or rule.

**Function**  
A reusable block of Python code that performs a task.

**Guardrail**  
A rule or check added to keep outputs safer, clearer, or more reliable.

**Histogram**  
A chart that shows how numeric values are distributed.

**Jupyter notebook**  
An interactive document made of code cells and text cells.

**JSON**  
A common structured data format that often looks like dictionaries and lists.

**Kernel**  
The part of a notebook environment that runs your code.

**LLM**  
Large language model. A system that can generate text from prompts.

**Logic bug**  
A mistake where code runs, but the result is wrong.

**Markdown**  
Formatted text used for notes, instructions, and headings in notebooks.

**Model**  
A system that uses data to help make predictions or decisions.

**Output**  
The result produced by code or an AI system.

**pandas**  
A Python library commonly used to work with tables of data.

**Plot**  
A chart used to visualize data.

**Prompt**  
The instructions or request given to an AI system.

**Response**  
The output returned from an API or AI system.

**Row**  
One record or example in a table.

**Scatter plot**  
A chart used to compare two numeric variables.

**Syntax error**  
An error caused by invalid Python structure.

**Tool**  
A function or capability that an AI workflow can call to perform a task.

**Workflow**  
A connected sequence of steps used to solve a task.
