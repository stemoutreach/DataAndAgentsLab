# Using AI Well

AI can be a powerful assistant for learning and problem solving. It can explain concepts, suggest approaches, help debug code, and help students move forward when they get stuck.

But AI is not autopilot.

The goal is not to let AI do all the thinking. The goal is to learn how to work **with** AI while still using your own judgment.

## The Main Idea

**AI helps generate. Humans must decide.**

That means students still need to:
- understand the problem
- choose a reasonable approach
- test the output
- check whether the answer makes sense
- explain what they changed and why

## Good Uses of AI

Good uses include:
- asking what an error message means
- asking for a simpler explanation of code
- asking for help breaking a task into steps
- asking how to test a solution
- comparing two possible approaches
- improving a weak prompt
- helping write a short explanation of results

## Weak Uses of AI

Weak uses include:
- pasting a full problem and copying the first answer
- submitting code you do not understand
- trusting confident-sounding output without checking it
- using AI instead of reading the instructions
- asking for the answer before trying to think through the task

## A Better Mindset

Instead of asking:
> Can AI do this for me?

Ask:
- What part do I understand already?
- What part is confusing me?
- What should I test first?
- What can AI explain or help me debug?
- How will I know if the answer is actually right?

## Good Questions To Ask AI

Better questions include:
- What is this code doing step by step?
- Why am I getting this error?
- Can you explain this in simpler language?
- What should I inspect first?
- I expected X but got Y. What could cause that?
- Can you help me debug this without giving the final answer?
- Can you compare these two approaches?

## Your Responsibility

Even when AI helps, the student is still responsible for:
- reading the task carefully
- understanding the code enough to use it
- testing the result
- checking the output
- noticing mistakes
- improving weak answers
- explaining the final solution

## Final Thought

A strong student is not the one who avoids AI completely.

A strong student is the one who can use AI wisely, think critically, test carefully, and still own the solution.
