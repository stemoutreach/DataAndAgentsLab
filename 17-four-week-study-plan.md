# 17 Four-Week Study Plan

This study plan helps students prepare over four weeks using the Data and Agents Lab repository.

The goal is not to read everything before starting.
The goal is to practice consistently, build confidence, and improve a little each week.

This plan is built around:
- Jupyter notebook practice
- Python and data skills
- debugging and testing
- classic AI / machine learning ideas
- agentic AI workflow ideas
- using AI as a helper, not autopilot

---

## How to Use This Plan

Each week includes:
- a main focus
- what to read
- what notebooks to complete
- what you should be able to do by the end of the week

Use this plan in a practical way:
1. open the notebook
2. try the task yourself
3. use AI only when needed
4. test your results
5. keep moving forward

You do not need to be perfect to make progress.

---

## Time Options

Choose the version that fits your schedule.

### Minimum Path
About **2–3 hours per week**

### Recommended Path
About **4–5 hours per week**

### Stretch Path
About **6+ hours per week**

If you are short on time, complete the **recommended notebooks first** and use the extra notebooks as practice where you need it most.

- Am I using AI to help me think, not replace thinking?

- thinking in connected steps

### Stretch Path

  - How is classic AI/data work different from agentic workflow work?

By the end of Week 4, you should be able to:
- write a prompt with role, task, constraints, and output format
- explain what a tool function does

## Reflection Questions

- Can I explain the difference between classic AI tasks and agentic AI tasks?
- Can I improve a weak prompt?
- Can I describe what information needs to be passed from one step to the next?

# If You Fall Behind

That is okay.

If you do not finish everything:
1. focus on the notebook sequence first
2. prioritize debugging and data basics
3. do at least one classic AI notebook and one agentic AI notebook
4. review your weak areas instead of trying to do everything quickly

It is better to understand fewer things well than rush through everything.
