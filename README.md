# Data and Agents Lab

A student-friendly training repo for learning by **doing** in Jupyter notebooks.

This repository helps students build skill in:
- Python
- Jupyter notebooks
- pandas and tabular data
- APIs and JSON
- debugging and testing
- plotting and interpretation
- classic AI / machine learning
- agentic AI
- using AI as a helper, not autopilot

## Start Here

Do these first:

1. Read [12-how-to-use-the-practice-notebooks.md](./12-how-to-use-the-practice-notebooks.md)
2. Open `notebooks/00-notebook-basics-practice.ipynb`
3. Start coding
4. Use the guides only when you need support or explanation

## Practice Notebooks

### 00 Notebook Basics Practice
Learn how to run cells, edit values, fix notebook order problems, and restart/rerun.

### 01 Python TODO Practice
Practice variables, lists, dictionaries, loops, functions, and starter-code editing.

### 02 pandas Data Practice
Load a CSV, inspect rows and columns, filter data, check missing values, and explain what you notice.

### 03 API and JSON Practice
Inspect JSON, navigate nested fields, and turn structured data into a table.

### 04 AI-Assisted Problem Solving Practice
Fix broken code, ask better AI questions, compare weak and better AI help, and verify your result.

### 05 Mini Classic AI Challenge
Use a small dataset to inspect patterns, define a rule, test edge cases, and explain your decisions.

### 06 Mini Agentic AI Challenge
Practice prompt structure, tool functions, connected steps, and output evaluation.


### 07 Debugging Practice
Find and fix different bug types: syntax, name, type, logic, and notebook-order issues.

### 08 Plotting and Interpretation
Create simple charts and explain what they show.

### 09 Agentic Workflow Practice
Use a mock LLM and a tool function to practice a small prompt-and-workflow loop.

### 10 Multi-Step Data Investigation
Work through a larger dataset from inspection to cleaning, charting, rule-building, and summary.
