# Data and Agents Lab

Data and Agents Lab is a student-friendly training repository for learning through hands-on work in Jupyter notebooks.

It is designed for:
- high school students
- beginner interns
- mentors and coaches

The repository helps students practice:
- Python basics
- notebook skills
- pandas and tabular data
- APIs and JSON
- debugging and testing
- plotting and interpretation
- classic AI / data investigation
- agentic AI workflow thinking
- using AI as a helper, not autopilot

## Start Here

1. Read [12-how-to-use-the-practice-notebooks.md](./12-how-to-use-the-practice-notebooks.md)
2. Read [17-four-week-study-plan.md](./17-four-week-study-plan.md)

4. Start practicing
